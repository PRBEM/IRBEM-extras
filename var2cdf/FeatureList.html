var2cdf features

The table below documents which features are available in which languages.
Feature Matlab IDL Applies To
Read structure var from cdf Mostly ? cdf2var
Write structure var to cdf Mostly ? var2cdf
Read standard numeric types from cdf Yes ? cdf2var
Write standard numeric types to cdf Yes ? var2cdf
Write logicals to cdf as short int Yes ? var2cdf
Write function handles to cdf as string Yes ? var2cdf
Write Matlab cell arrays / IDL collections to cdf as multiple entries var{i} Yes ? var2cdf
Write structure arrays to cdf as multiple entries var(i).subvar Yes ? var2cdf
Write arrays of structures to cdf as multiple entries var(:).subvar No ? var2cdf
Substitute " "(space) for empty string YES ? var2cdf
Concatenate cell array of strings to string matrix (option) Yes, by default N/A? var2cdf
Convert numeric types to double (option) Yes N/A cdf2var
Convert .mat file to cdf Yes N/A mat2cdf
Load variables to caller namespace when no output argument specified No ? cdf2var
